Hangman
-------
Author: Karan Tamhane
Course: Introduction to Computer Science and Programming
School: MITx (edX)
Problem Set: 3

Summary:
	Implements the word game 'Hangman' in which a user is supposed to guess a secret word in a fixed number of turns by guessing one character per turn. Word is displayed with correctly guessed letters exposed, while other letters remain undisclosed. Game ends when word is guessed or turns are exhausted.

Source code: hangman.py

Modules imported:
	string, random

Files:
	"words.txt" - loads valid wordlist

Usage:

	Program must be run in a python shell.
	
	Call hangman() to play a game.
