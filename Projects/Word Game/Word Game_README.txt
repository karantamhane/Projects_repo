6.00x Word Game
---------------
Author: Karan Tamhane
Course: Introduction to Computer Science and Programming
School: MITx (edX)
Problem Set: 4

Summary:
	Implements a 'Boggle'-like word game. User is to create valid english words from a given set or 'hand' of characters. Words are scored based on values of constituent characters like in scrabble. E.g. - If given hand is 'ghwelusol', user can make 'hello' and be left with 'gwus' or make 'glows' and be left with 'helu'

Source code: word_game.py

Modules imported:
	random, string

Files:
	
	"words.txt" - loads a valid wordlist

Usage:
	
	Program must be run in a python shell.

	Call playGame() to play the game.
