Pong
----
Author: Karan Tamhane
Course: Interactive Programming in Python
School: Rice University (Coursera)
Problem Set: 8

Summary:
	Pong is a two-player game in which players bounce a ball off of paddles and defend their own sides.

Source code: pong.py

Usage:

	Program must be run in a web browser.
	Navigate to http://www.codeskulptor.org/#user4-qkjh50WrwV5FW8j.py
	Click the 'Run' button (or play button) at the top left corner of the page.

	Left hand player uses 'w' and 's' keys to move paddle up and down respectively
	Right hand player uses 'up arrow' and 'down arrow' keys to move paddle up and down respectively
