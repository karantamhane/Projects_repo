Asteroids
---------
Author: Karan Tamhane
Course: Interactive Programming in Python
School: Rice University (Coursera)
Problem Set: 8

Summary:
	Asteroids is a version of the classic game involving a spaceship shooting asteroids in space.

Source code: asteroids.py

Usage:

	Program must be run in a web browser.
	Navigate to http://www.codeskulptor.org/#user17_WOwCBzNUBjpqYau_4.py
	Click the 'Run' button (or play button) at the top left corner of the page.
	
	Use arrow keys to turn and accelerate ship and spacebar to shoot.
	Players get three lives and a player 'dies' if an asteroid hits his ship.
	Scoring is based on number of asteroids a player shoots.
